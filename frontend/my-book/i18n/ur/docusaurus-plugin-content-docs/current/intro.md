# ٹیوٹوریل تعارف

آئیے **5 منٹ سے کم میں Docusaurus** کو دریافت کریں۔

## شروع کرنا

**نئی سائٹ بنا کر** شروع کریں۔

یا **[docusaurus.new](https://docusaurus.new)** کے ساتھ **فوری طور پر Docusaurus آزمائیں**۔

### آپ کو کیا چاہیے ہوگا

- [Node.js](https://nodejs.org/en/download/) ورژن 20.0 یا اس سے اوپر:
  - Node.js انسٹال کرتے وقت، آپ کو انحصارات سے متعلق تمام چیک باکسز کو چیک کرنے کی سفارش کی جاتی ہے۔

## نئی سائٹ بنائیں

**کلاسک ٹیمپلیٹ** استعمال کرتے ہوئے نئی Docusaurus سائٹ بنائیں۔

کلاسک ٹیمپلیٹ خودکار طور پر آپ کے پروجیکٹ میں شامل ہو جائے گا جب آپ کمانڈ چلائیں گے:

```bash
npm init docusaurus@latest my-website classic
```

آپ اس کمانڈ کو Command Prompt، Powershell، Terminal، یا اپنے کوڈ ایڈیٹر کے کسی دوسرے مربوط ٹرمینل میں ٹائپ کر سکتے ہیں۔

یہ کمانڈ Docusaurus چلانے کے لیے ضروری تمام انحصارات بھی انسٹال کرتی ہے۔

## اپنی سائٹ شروع کریں

ڈیولپمنٹ سرور چلائیں:

```bash
cd my-website
npm run start
```

`cd` کمانڈ اس ڈائرکٹری کو تبدیل کرتی ہے جس کے ساتھ آپ کام کر رہے ہیں۔ اپنی نئی بنائی گئی Docusaurus سائٹ کے ساتھ کام کرنے کے لیے، آپ کو ٹرمینل کو وہاں نیویگیٹ کرنا ہوگا۔

`npm run start` کمانڈ آپ کی ویب سائٹ کو مقامی طور پر بناتی ہے اور اسے ڈیولپمنٹ سرور کے ذریعے پیش کرتی ہے، جو آپ کے لیے http://localhost:3000/ پر دیکھنے کے لیے تیار ہے۔

`docs/intro.md` (یہ صفحہ) کھولیں اور کچھ لائنیں ایڈٹ کریں: سائٹ **خودکار طور پر دوبارہ لوڈ ہوتی ہے** اور آپ کی تبدیلیاں ظاہر کرتی ہے۔
